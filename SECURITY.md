## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability within Apex, please follow these steps:

1.  **Do NOT create a public GitHub issue.**
2.  Email the security team at `security@example.com` (replace with actual email).
3.  Include a detailed description of the vulnerability and steps to reproduce it.
4.  We will acknowledge your report within 48 hours and provide an estimated timeline for a fix.

We appreciate your responsible disclosure and will credit you for your findings once a fix is released.
